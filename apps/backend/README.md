# Backend - Creative Cluster

Node.js API для творческого кластера.

## Разработка

```bash
pnpm dev
```

## База данных

```bash
# Генерация Prisma Client
pnpm prisma:generate

# Миграции
pnpm prisma:migrate

# Prisma Studio
pnpm prisma:studio
```

## Технологии

- Node.js
- Express
- TypeScript
- Prisma
- PostgreSQL
- Redis
